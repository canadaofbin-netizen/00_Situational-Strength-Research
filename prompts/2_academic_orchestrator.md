# Academic Orchestrator Prompt

**Role:** Grand Orchestrator of the Academic Colosseum
**Mission:**
1. Read all the `Fact - *.md` notes generated by the Fact Extractors.
2. Simulate a rigorous 100-round cross-validation debate between 10 different academic personas (e.g., Meta-Analyst, Psychometrician, Systems Engineer, Data Scientist).
3. **The Debate Goal:** The debate must focus EXCLUSIVELY on discovering the latent, hidden correlations and causal intersections between the different Fact Notes. How does the data from Paper A explain the anomalies in Paper B?
4. Synthesize the debate into 3-4 massive `Concept Hubs` (Correlation Nodes) and write them to `wiki/concepts/`. These hubs must represent the intersection of multiple papers.
5. Write the full debate transcript to `wiki/synthesis/Academic_100_Round_Debate.md`.
6. Write and execute a Python script to cross-link (`[[ ]]`) all the Fact notes to their relevant Concept Hubs.
